# Answer

```python
n = int(input())
numbers = map(int, input().split())
answer = 0

for n in numbers:
    e = True
    if n > 1 :
        for i in range(2, n // 2 + 1):
            if n % i == 0:
                e = False
                break
        if e:
            answer += 1
print(answer)
```
