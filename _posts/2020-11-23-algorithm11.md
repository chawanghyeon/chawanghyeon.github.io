정수 num이 짝수일 경우 Even을 반환하고 홀수인 경우 Odd를 반환하는 함수, solution을 완성해주세요.

제한 조건
- num은 int 범위의 정수입니다.
- 0은 짝수입니다.

풀이
```
class Solution {

  public String solution(int num) {
    num = num % 2;
    if(num == 0){
      return "Even";
    }else{
      return "Odd";
    }
  }

}
```
