# Answer

```python
dp = [1, 1, 1, 2, 2, 3, 4, 5, 7, 9]

for _ in range(int(input())):
    n = int(input())
    
    if n <= 10:
        print(dp[n - 1])
    else:
        dp = dp + [0] * n
        
        for i in range(10, n + 1):
            dp[i] = dp[i-1] + dp[i-5]
            
        print(dp[n-1])
```
