# Answer

```python
word = input()
answer = len(word)

croatia = ['c=', 'c-', 'dz=', 'd-', 'lj', 'nj', 's=', 'z=']

for i in croatia:
    if i in word:
        for j in range(word.count(i)):
            idx = word.find(i)
            word_lenth = len(i)
            answer = answer - word_lenth + 1
            word = word[0:idx] + '*' + word[idx + word_lenth:]

print(answer)
```
