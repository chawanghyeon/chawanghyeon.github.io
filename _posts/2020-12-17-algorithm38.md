```java
import java.util.HashSet;

class Solution {

    public int solution(int[] nums) {
        int length = nums.length / 2;
        
        HashSet<Integer> set = new HashSet<Integer>();
        
        for(int i : nums){
            set.add(i);
        }
        
        int setSize = set.size();
        
        if(length < setSize){
            return length;
        }
        return setSize;
    }
    
}
```
HashSet을 이용하면 쉽게 풀 수 있습니다.
