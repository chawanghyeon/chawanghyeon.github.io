단계별로 풀어보기의 기본 수학1 파트입니다.  
이 게시물은 제가 문제를 풀 때 마다 업데이트 할 예정입니다.  

## 손익분기점 - 1712번
```python
a, b, c = map(int, input().split())
n = 1
if b >= c:
    print(-1)
else:
    print(a // (c - b) + 1)
```
## 벌집 - 2292번
```python
n = int(input())
d = 1
while n > 1:
    n -= (6 * d)
    d += 1
print(d)
```
## 분수찾기 - 1193번
```python
x = int(input())
line = 1

while x > line:
    x -= line
    line += 1
    
if line % 2 == 0:
    a = x
    b = line - x + 1
else:
    a = line - x + 1
    b = x
```
