단계별로 풀어보기의 문자열 파트입니다.  
이 게시물은 제가 문제를 풀 때 마다 업데이트 할 예정입니다.  

## 아스키 코드 - 11654번
```python
print(ord(input()))
```
## 숫자의 합 - 11720번
```python
n = input()
a = input()
answer = 0

for i in a:
    answer += int(i)
    
print(answer)
```
## 알파벳 찾기 - 10809번
```python
s = input()
a = [-1] * 26

for i, v in enumerate(s):
    if a[ord(v) - 97] == -1:
        a[ord(v) -97] = i

for i in a:
    print(i)
```
## 문자열 반복 - 2675번
```python
for _ in range(int(input())):
    idx, word = input().split()
    for i in word:
        print(i * int(idx), end = '')
    print()
```
## 단어 공부 - 1157번
```python
word = input().upper()
a = []

for i in range(65, 91):
    a.append(word.count(chr(i)))
    
print('?' if a.count(max(a)) > 1 else chr(a.index(max(a)) + 65))
```
