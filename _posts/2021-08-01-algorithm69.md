# Answer

```python
n = int(input())
c = 2
m = 1

while True:
  if m < n:
    m += c
    c += 1
  else :
    if (c % 2) == 0:
      print(str(1+(m-n))+"/"+str((c-1) - (m-n)))
      break
    else:
      print(str((c-1) - (m-n))+"/"+str(1+(m-n)))
      break
```

```python
n = int(input())
line = 1

while n > line:
    n -= line
    line += 1
    
if line % 2 == 0:
    a = n
    b = line - n + 1
else:
    a = line - n + 1
    b = n
    
print(a, '/', b, sep='')
```
