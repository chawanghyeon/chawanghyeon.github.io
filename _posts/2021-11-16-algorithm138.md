# Answer

```python
dp = []

for i in range(1, 32) :
    dp.append([1] * i)

for i in range(2, 31) :
    for j in range(1, i) :
        dp[i][j] = (dp[i-1][j-1] + dp[i-1][j])

for i in range(int(input())):
    n, k = map(int, input().split())
    print(dp[k][n])
```
